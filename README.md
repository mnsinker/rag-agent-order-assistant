# RAG Agent Order Assistant

An AI-powered customer support assistant that:
- Retrieves refund policies (RAG)
- Checks order status
- Decides whether a refund is allowed (Agent + Tools)

## Features
- RAG-based policy retrieval
- Tool-based decision system
- Modular architecture (agent / tools / llm)

## Tech Stack
- Python
- OpenAI API
